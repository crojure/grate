# Grate

Guarantee Rate Homework
Step #1 - Command line app

## Usage

`lein run [file-location]`

OR 

```
lein uberjar
java -jar target/uberjar/grate-0.1.0-SNAPSHOT-standalone.jar [file-location]
```
